
# logfmt

This package implements a fast SAX-style [logfmt](https://brandur.org/logfmt) parser via the almighty Ragel FSM generator.

## Installation

```
$ clib install logfmt
```

## Benchmarks

 Non-scientific results from my MBP with 122b logs:

 ```
len: 122
ops: 10,000,000
ops/s: 4,155,844.25
duration: 2.41s
```
